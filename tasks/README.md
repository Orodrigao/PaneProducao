# Planos de tarefa

Não existe um `todo.md` global.

Para tarefa complexa, o plano aprovado pode ser registrado em arquivo com nome
específico da funcionalidade ou no corpo do draft PR. Ele não vira fonte do
estado geral do projeto.

Ao concluir:

- mantenha o plano somente se explicar decisão não óbvia;
- use o PR para histórico de execução;
- atualize `docs/CURRENT_STATE.md` apenas se o estado material mudou.
