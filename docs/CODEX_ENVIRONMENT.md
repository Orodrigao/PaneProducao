# Ambiente Codex — arquivo legado

O ambiente não é fonte de estado do projeto.

Antes de trabalhar:

1. abra o checkout correto;
2. rode `git status -sb`;
3. atualize referências remotas;
4. crie a branch da tarefa a partir do `origin/main` atual;
5. siga [../AGENTS.md](../AGENTS.md) e
   [CURRENT_STATE.md](CURRENT_STATE.md).

Não reutilize caminhos, tokens, comandos tolerantes a erro ou snapshots de
sessões antigas. O histórico deste arquivo permanece no Git.
